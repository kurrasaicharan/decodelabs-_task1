## 📌 Project Overview

The **Python To-Do List Application** is a simple command-line based task management program built using Python.

This project helps users manage their daily activities by allowing them to add new tasks, view existing tasks, remove completed tasks, and exit the application through an interactive menu.

It is designed to practice fundamental Python concepts such as lists, loops, conditional statements, and user input handling.

---

## 🚀 Features

✨ Add new tasks  
✨ Display all saved tasks  
✨ Remove completed tasks  
✨ Interactive menu system  
✨ Simple and beginner-friendly interface  

---

## ▶️ How to Run

1. Install Python on your system.

2. Clone this repository:

```

git clone YOUR_REPOSITORY_LINK

```

3. Open the project folder:

```

cd To-Do-List

```

4. Run the program:

```

python todo_list.py

```
